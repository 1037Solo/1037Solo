# 贡献指南

[English](./CONTRIBUTING.md) | [简体中文](./CONTRIBUTING.zh-CN.md)

感谢你对 1037Solo 的关注！

## 关于本仓库

这是 1037Solo 团队的 GitHub 官方主页仓库，作为公开落地页使用，**不是**功能开发仓库。

本仓库接受的贡献范围：

- 事实信息的纠正；
- 文档表述的改进；
- 翻译质量的提升；
- 路线图或生态方向的建议（通过 Issue）。

## 如何贡献

### 提交 Issue

- 使用 [GitHub Issues](https://github.com/1037Solo/1037Solo/issues) 报告问题或提出建议。
- 提交前请先检查是否已有相同 Issue。
- 请提供清晰的上下文，如适用请附上相关章节链接。

### 提交变更

1. Fork 本仓库。
2. 从 `main` 创建功能分支（`git checkout -b fix/description`）。
3. 完成修改，如涉及双语内容请同步更新中英文版本。
4. 使用清晰的提交信息，遵循 [Conventional Commits](https://www.conventionalcommits.org/) 规范（如 `docs: 修复 README 中的断链`）。
5. 提交 Pull Request 并附上变更说明。

### 合作与共建

如需生态合作、学术共建或其他协作，请在 Issue 中使用 **collaboration** 标签。

## 行为准则

所有参与者须遵守我们的[行为准则](./CODE_OF_CONDUCT.md)。

## 有疑问？

请通过 [support@1037solo.com](mailto:support@1037solo.com) 联系我们，或直接提交 Issue。

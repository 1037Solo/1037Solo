# Contributing to 1037Solo

[English](./CONTRIBUTING.md) | [简体中文](./CONTRIBUTING.zh-CN.md)

Thank you for your interest in contributing to 1037Solo!

## About This Repository

This is the official GitHub profile repository for the 1037Solo team. It serves as a public landing page and is **not** a feature development repository.

Contributions here are limited to:

- Corrections to factual information;
- Improvements to documentation clarity;
- Translation improvements;
- Suggestions for roadmap or ecosystem direction (via Issues).

## How to Contribute

### Reporting Issues

- Use [GitHub Issues](https://github.com/1037Solo/1037Solo/issues) to report problems or suggest improvements.
- Check existing issues before creating a new one.
- Provide clear context and, if applicable, links to the relevant section.

### Submitting Changes

1. Fork this repository.
2. Create a feature branch from `main` (`git checkout -b fix/description`).
3. Make your changes and ensure both English and Chinese versions are updated if applicable.
4. Commit with a clear message following [Conventional Commits](https://www.conventionalcommits.org/) (e.g., `docs: fix broken link in README`).
5. Open a Pull Request with a clear description of the change.

### Collaboration and Partnerships

For ecosystem partnerships, academic co-creation, or broader collaboration proposals, please open an Issue with the **collaboration** label.

## Code of Conduct

All participants are expected to follow our [Code of Conduct](./CODE_OF_CONDUCT.md).

## Questions?

Reach out via [support@1037solo.com](mailto:support@1037solo.com) or open an Issue.

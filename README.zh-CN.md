# 1037Solo

[English](./README.md) | [简体中文](./README.zh-CN.md)

1037Solo 团队的 GitHub 官方主页仓库。

## 使命

1037Solo 致力于构建聚焦学习场景、学术效率与学生知识工作流的 AI 原生产品。

我们的目标是帮助学生把碎片信息转化为结构化理解、可执行产出与长期学习成长。

## 本仓库定位

这个特殊仓库（`1037Solo/1037Solo`）是团队在 GitHub 的官方公开落地页。

主要用于：
- 官方定位与对外沟通；
- 产品与仓库导航；
- 路线图与更新日志的公开发布。

本仓库不作为功能开发仓库使用。

# Sponsors

[English](./SPONSORS.md) | [简体中文](./SPONSORS.md#赞助支持)

1037Solo is a self-funded, independent project maintained by a solo developer. Every contribution directly supports server infrastructure, AI API credits, and sustained product development.

## Why Sponsor

- Keep 1037Solo's core products free and accessible for students.
- Support an independent developer building AI-native tools for campus learning.
- Help maintain reliable infrastructure (servers, databases, domain renewals).
- Contribute to the open-source ecosystem around StudySolo.

## Sponsorship Tiers

| Tier | Contribution | Benefits |
|---|---|---|
| ☕ **Supporter** | Any amount | Name listed in Sponsors section; our sincere gratitude |
| 🌟 **Backer** | ¥50+ / month or equivalent | Name + link listed; priority beta access to new products |
| 💎 **Patron** | ¥200+ / month or equivalent | Logo displayed on landing page and README; direct communication channel with the developer; early access to all products and API capabilities |
| 🏢 **Institutional Sponsor** | Custom | Prominent logo placement across all 1037Solo properties; co-branding opportunities; dedicated architecture consultation; custom integration support |

> Amounts are indicative. We appreciate contributions of any size.

# 赞助支持

1037Solo 是由一名开发者独立维护的自费项目。每一份赞助都将直接用于服务器基础设施、AI API 额度与持续产品开发。

## 为什么赞助

- 帮助 1037Solo 的核心产品持续免费开放给学生群体。
- 支持一位独立开发者构建面向校园学习的 AI 原生工具。
- 帮助维持可靠的基础设施（服务器、数据库、域名续费）。
- 为 StudySolo 相关的开源生态做出贡献。

## 赞助层级

| 层级 | 贡献 | 权益 |
|---|---|---|
| ☕ **支持者** | 任意金额 | 姓名列入赞助者名单；我们真诚的感谢 |
| 🌟 **资助者** | ¥50+ / 月或等值 | 姓名 + 链接展示；新产品优先内测资格 |
| 💎 **赞助人** | ¥200+ / 月或等值 | Logo 展示于落地页与 README；与开发者的直接沟通渠道；所有产品与 API 能力的抢先体验 |

> 金额仅供参考，我们感谢任何规模的贡献。
